# OPTIMA Sleep Quality Analysis

This code is intended for use with data from the Operationalizing Digital PhenoTyping in the Measurement of Anhedonia (OPTIMA) study conducted at UCLA.


## Key file descriptions
1. sleep_quality_analysis.ipynb: Main notebook for analyses. Generates plots and tables utilized in result reporting.
2. pyproject.toml: Use this file to load dependencies using a tool such as poetry. Note dependence on my other github repositories for feature generation and plotting.
